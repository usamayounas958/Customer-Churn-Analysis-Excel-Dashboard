This project focuses on analyzing customer churn behavior using Microsoft Excel. The goal is to identify churn patterns, revenue impact to support data-driven retention strategies. The analysis uses Pivot Tables, calculated fields, KPIs, and interactive dashboards to transform raw customer data into actionable business insights.

Business Problem

Customer churn directly impacts revenue and long-term growth.
This project answers key business questions such as:

What percentage of customers have churned?
Which regions and segments experience the highest churn?
Which customer segments cause the highest revenue loss?
Is churn increasing over time?

Dataset Description

The dataset includes customer-level information such as:

Customer ID
Join Date
Last Purchase Date
Region
Customer Segment
Total Revenue

Derived metrics were created to support churn analysis.

Data Preparation & Feature Engineering

Performed the following steps in Excel:

Converted raw data into structured tables
Created calculated columns:
Days Since Last Purchase
Customer Tenure
Churn Status (Active / Churned)
Applied correct date and numeric formatting
Used logical formulas (IF, TODAY, DATEDIF)

Key KPIs

The dashboard tracks the following KPIs:

Total Customers
Churn Rate (%)
Retention Rate (%)
Revenue Lost Due to Churn

Dashboard Features

The interactive Excel dashboard includes:

Churn Status by Region
Churn Distribution by Customer Segment (%)
Revenue Loss by Segment
Monthly Churn Trend Analysis

Key Insights

50% of customers have churned, indicating a high retention risk
Small Business segment shows the highest churn share (40%)
Corporate segment contributes the highest revenue loss, despite lower churn count
Churn shows an upward trend in later months, signaling retention issues over time
